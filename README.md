# Fluxo — Solicitação de Liberação Temporária de Usuários Adicionais

Fluxo em **HTML + SVG + CSS + JS puro** para padronizar a solicitação por e-mail à Central de Projetos Sankhya.

- Destinatário: `centraldeprojetos@sankhya.com.br`
- Assunto: `LIBERAÇÃO DE USUÁRIOS: Apoio a Consultor`
- Corpo: modelo com campos (➡️) pronto para copiar e preencher

## Arquivos
- `index.html` (raiz do repositório)
